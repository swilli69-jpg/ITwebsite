What is Pure Data?

Pure Data (Pd) is an open-source visual programming language designed for creating interactive computer music, sound design, and multimedia projects. Instead of writing lines of code, users connect objects with patch cords in a graphical interface to process and generate sound, MIDI, and visual data in real-time.

How it Works

Pd uses a dataflow model, meaning that data — such as control messages, audio signals, or numerical values — flows between interconnected objects. Each object performs a specific function, such as generating a tone, filtering sound, or controlling playback.

[osc~ 440]
|
[*~ 0.2]
|
[dac~]

This example creates a 440 Hz sine tone, multiplies it by 0.2 (reducing amplitude), and sends it to your audio output.

Why Use Pd?

  • Real-time Audio Processing – Build and perform interactive sound systems live.
  • Extensible – Hundreds of libraries extend Pd for visuals, sensors, and more.
  • Educational – Great for learning digital signal processing and computer music.
