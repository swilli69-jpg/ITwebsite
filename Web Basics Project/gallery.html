Patch Gallery

A collection of Pure Data patches I've created for sound design, composition, and experimentation. Feel free to explore and adapt these for your own projects!

Granular Delay

[adc~]
|
[delwrite~ grain 5000]
|
[delread~ grain]
|
[*~ 0.7]
|
[dac~]

A real-time granular delay effect that creates evolving textures from incoming audio. Adjustable grain size and feedback.

audio fx live input

Step Sequencer

[metro 500]
|
[counter 0 15]
|
[tabread notes]
|
[mtof]
|
[osc~]

16-step MIDI sequencer with variable tempo. Perfect for creating rhythmic patterns and melodic loops.

sequencer composition

Ambient Drone Generator

[osc~ 165]
[osc~ 220]
| | |
[+~]
|
[lop~ 800]

Generates rich harmonic drones using multiple oscillators. Includes low-pass filtering for warm, atmospheric sounds.
